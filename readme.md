# Aroma & Grano – Sitio Web Cafetería

Landing page informativa para la cafetería **Aroma & Grano**, orientada a presentación visual, productos y contacto rápido con clientes.

## Descripción

El sitio incluye:
- Slider de imágenes
- Listado de productos
- Menú descargable en PDF
- Botón de contacto vía WhatsApp
- Estructura de top y footer optimizada

El trabajo se centró en **ordenar, simplificar y mejorar la mantenibilidad**, sin alterar la lógica existente.

## Cambios principales

- Consolidación de **6 archivos CSS en un solo `styles.css`**.
- Reorganización visual del **top** y **footer**.
- Integración de imágenes para slider y productos.
- Implementación de botón de **WhatsApp**.
- Creación y enlace del **menú `menu-cafeteria.pdf`**.

## Alcance

- No se modificó la lógica JavaScript.
- No se alteró contenido funcional previo.
- Cambios enfocados en estructura y presentación.

## Revisión sugerida

- Vista móvil.
- Funcionamiento del slider.
- Descarga del PDF.
- Comportamiento del botón de WhatsApp.
